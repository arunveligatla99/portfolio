## Problem

Clinical and policy content has two non-negotiables: every claim needs a citation, and every citation has to point to a passage that actually supports the claim. A naive RAG pipeline will hallucinate citations, retrieve the wrong chunk, or quietly degrade as the corpus grows. Worse, it will pass smoke tests while quietly breaking on the long tail.

## Approach

Treat the RAG pipeline as a regular software system with regular software guarantees: contracts, tests, and CI gates.

- **Hybrid retrieval.** BM25 for lexical recall on policy ids and proper nouns, dense vectors for semantic recall, cross-encoder rerank for the top of the funnel. Each stage is independently swappable and benchmarked.
- **Grounded generation.** GPT-4o with a strict citation schema, a refusal path when the retrieved context does not support an answer, and a prompt registry that versions every template.
- **Continuous evaluation.** RAGAS metrics (faithfulness, answer relevance, context precision, context recall) on a curated gold set, run on every PR. A regression on any metric fails CI.
- **Replayability.** Every production trace is logged with the prompt version, retrieval candidates, scores, and final answer, so any output can be reconstructed and audited.

## Stack

- **API:** Python with FastAPI.
- **Models:** Azure OpenAI GPT-4o for generation, text-embedding-3-large for retrieval.
- **Index:** pgvector with HNSW.
- **Eval:** RAGAS, custom faithfulness checker over citation spans.
- **CI:** GitHub Actions with a dedicated eval job that uses a frozen gold set.
- **Tracing:** OpenTelemetry, vendor-neutral; logs replayable.

## Outcomes

- 92.5 percent citation accuracy on the held-out gold set.
- p95 end-to-end latency under 800ms with the hybrid pipeline.
- 167 passing tests, including RAGAS evals, all gating CI.
- Zero shipped regressions on the citation metric over the measurement window because the gate prevented them.

## Lessons

- Treat the eval set as production data: version it, review it like code, never overwrite it.
- Faithfulness is a different metric from "looks right". A model that paraphrases the source confidently can still pass casual review and fail the metric.
- A refusal path is cheaper than a wrong answer with a citation.
